  • ➕ Add Reactant: Adds more reactant, so the equilibrium shifts right (toward products) to use up the extra reactant.
  • 🌡️ Increase Temp: If the reaction is exothermic (heat is a product), it shifts left. If endothermic (heat is a reactant), it shifts right.
  • 🔽 Increase Pressure: Shifts the equilibrium toward the side with fewer gas molecules. If both sides have equal gas moles (or no gases), there's no shift.
